到家MOCK平台

到家MOCK平台是针对项目前后端分离开发模式，供FE及RD等相关人员对接口文档的一个存储和查看平台，待相关RD同学填写完接口数据格式后，平台提供MOCK链接供FE同学在开发阶段作为接口参考数据

MOCK平台功能

集群相关

  • 创建集群，填写相关信息
  • 集群列表查看所有已填写集群信息
  • 集群详情页展示接口具体信息，可编辑保存

接口相关

  • 接口列表分页展示已有接口列表
  • 接口列表支持多项搜索功能
  • 新建接口必须关联集群
  • 同一个集群下，接口名称不能重复
  • 接口详情页展示接口具体信息，可编辑保存

实体相关

  • 实体列表展示已定义实体列表
  • 实体列表支持集群及实体名称搜索
  • 同一个集群下，实体名称不能重复
  • 实体详情页面展示实体具体信息，可编辑保存
